SLIDE 2: PROBLEM STATEMENT
--------------------------------------------------------------------------------
• Diabetes affects millions globally, including Nigeria
• Many cases undiagnosed until complications arise
• Traditional screening is time-consuming and resource-intensive
• Need for automated, efficient screening tools

SLIDE 3: OBJECTIVES
--------------------------------------------------------------------------------
1. Develop three ML algorithms for diabetes classification
2. Compare performance of Naïve Bayes, SVM, and Decision Tree
3. Identify the most effective algorithm
4. Provide recommendations for clinical implementation

SLIDE 4: METHODOLOGY
--------------------------------------------------------------------------------
Dataset: 768 patients, 8 clinical features
Process:
  ① Data Collection → ② Preprocessing → ③ Model Training →
  ④ Hyperparameter Optimization → ⑤ Evaluation → ⑥ Comparison

SLIDE 5: DATA PREPROCESSING
--------------------------------------------------------------------------------
• Handled missing values (median imputation)
• Split data: 80% training, 20% testing
• Feature scaling (StandardScaler)
• Result: Clean, normalized dataset ready for ML

SLIDE 6: MODELS DEVELOPED
--------------------------------------------------------------------------------
1. Naïve Bayes: Fast, probabilistic
2. SVM: Margin-based, 48 hyperparameter combinations tested
3. Decision Tree: Rule-based, 160 hyperparameter combinations tested

SLIDE 7: RESULTS - PERFORMANCE COMPARISON
--------------------------------------------------------------------------------
Model           Accuracy     Precision    Recall       F1-Score    
--------------------------------------------------------------------------------
Naive Bayes          70.13%      0.5667      0.6296      0.5965
SVM                  69.48%      0.5778      0.4815      0.5253
Decision Tree        68.83%      0.6364      0.2593      0.3684

SLIDE 8: BEST MODEL
--------------------------------------------------------------------------------
🏆 Winner: Naive Bayes

Performance:
  • Accuracy: 70.13%

Why This Model?
  • Best balance between precision and recall
